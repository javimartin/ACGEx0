SceneDescription Tutorial
 


This tutorial will show you how you can define a Scene for the RayTracer.

1. Begin

The scene is described using the XML language. To begin the scene tag has to be opened.

 
 
<Scene background="0.0 0.0 0.0 1.0" ambient=".02 .02 .02 1.0" refractionIndex="1.0">


</Scene>
 

 
The attribute background defines the background color of the scene. ambient defines the scene's ambient parameter used for phong lighting. The refraction index of the scenes medium (i.e "air") is set by refractionIndex. The refraction value is a real number unequal zero. The colors are clamped to [0,1].



2. Camera


The first thing to be defined in the scene is the camera.

 
 
<Scene ...>

<Camera position="0.0 0.0 0.0" direction="0.0 0.0 -1.0" up="0.0 1.0 0.0" angle="16" resolutionX="300" resolutionY="300"/>


</Scene>
 

 
position, direction and up define the orientation of the camera in space. The angle attribute defines the opening angle of the camera in degrees. The two values resolutionX and resolutionY are the image resolutions used by the camera film. resolutionX and resolutionY of type integers must be defined, the other attributes are optional.



3. Material and Texture definition

Now materials and textures can be defined. Each of them are loaded with their names in order to assign to elements later.

 
 
<Scene ...>
<Camera .../>

<Materials>
 <color_red emission="0.0 0.0 0.0 1.0" ambient="1.0 0.0 0.0 1.0" diffuse="1.0 0.0 0.0 1.0" specular="0.2 0.0 0.0 1.0" shinyness="10.0"/>
 ...
<Materials/>

<Textures>
 <texture_one file="filename.tga"/>
 ...
<Textures/>

</Scene>
 

 
color_red and texture_one are the names of the material and texture respectively. The values emission, ambient, diffuse, specular and shinyness are the parameters of the material. All of them have to be defined. Otherwise the material will not be loaded. As for the texture, the file attribute is the name of the texture to be loaded and its position is relative to the scene description file. It also must be defined. The image file format is a 24-bit targa (.tga).



4. Elements

The final step is to create the geometry. These should be defined after the materials and textures as they can have references to to them. All elements of the scene can be defined inside the <Elements> Tag

 
 
<Scene ...>

...

<Textures/>

<Elements>

<Element_name ... >
 <Material ref="color_red"/>
 <Texture ref="texture_one"/>
<Element_name/>

...

<Elements>

</Scene>
 
 
The example above shows how to reference textures and materials. color_red and texture_one are the names you defined before. If there is no material or texture inside the elements tags, or the name is not found the default material will be choosen and no texture will be used.
